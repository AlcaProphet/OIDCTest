常用 Scope 说明

openid 必须，启用 OIDC 认证，获取 ID Token
profile 用户基本信息：姓名、用户名、头像等
email 邮箱地址及是否已验证
offline_access 获取 Refresh Token，支持长期离线刷新
roles 用户角色信息
groups 用户所属组
phone 电话号码
address 地址信息
多个 scope 用空格分隔，如 openid profile email
